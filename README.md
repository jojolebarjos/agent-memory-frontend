# Agent Memory - Front-end

This repository provides a simple Web interface for agentic memory experiments.


## Getting Started

This project relies on [`npm`](https://www.npmjs.com/) and [Vite](https://vite.dev/).
You can install the project dependencies using:

```sh
npm install
```

Assuming the WebSocket backend is running on `ws://localhost:8080` (cf. [`vite.config.ts`](./vite.config.ts)), you can then start the development server using:

```sh
npm run dev
```
